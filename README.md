# Setup

You must have nodejs and npm installed for this to work

## Install grunt
	$npm install -g grunt-cli

## Setup
	$npm install && bower install && foundation update && grunt

## Development
To watch for Sass changes:

	$grunt watch

## Run
	$npm start
